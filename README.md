# ChronosGuessr

A personalized location and date guessing game built as a birthday present. Players try to guess where and when photos were taken at various locations in California.

## Features

- Photo display with location and date guessing interface
- Precise scoring system
- Progress tracking
- Personalized California locations
- Beautiful, modern UI

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## How to Play

1. A photo will be displayed
2. Use the interface to guess the location and date when the photo was taken
3. Submit your guess
4. See how close you were to the actual location and date
5. Track your progress through multiple rounds

## Technical Stack

- React
- TypeScript
- Vite
- Chakra UI
- date-fns
