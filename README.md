# chronosguessr

An open-source [TimeGuessr](https://timeguessr.com) clone, with photos pulled from spots around Northern California. Players guess both where and when the photo was taken. Originally built as a birthday present. Available at https://esawadhana.github.io/chronosguessr/.

Vite + React + TypeScript, Chakra UI, Leaflet for the maps, Framer Motion for transitions.
